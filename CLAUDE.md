## REGLAS DE CÓDIGO OBLIGATORIAS
- Un solo archivo HTML cuando sea posible (más fácil de subir y vender)
- Todo el código comentado en español
- Diseño responsivo — primero móvil, luego desktop
- Colores: paleta profesional de construcción (azul oscuro, naranja, blanco)
- Sin dependencias externas si es posible
- Tiempo de carga menor a 2 segundos
- Funciona sin internet después de cargar (offline-first)
- Probado en Chrome Android antes de declarar listo

## REGLAS DE DISEÑO
- Apariencia: profesional, moderna, confiable
- NO usar plantillas genéricas de Bootstrap
- Tipografía: Inter o system fonts (no Google Fonts para evitar dependencias)
- Iconos: emojis o SVG inline (sin librerías externas)
- Cada producto debe verse como software de $500+ MXN aunque cueste $99
